How to Use Cross-Domain Knowledge Graph
Single Upload:

Upload one file to create a knowledge graph within a single domain.

Cross-Domain Upload:

Upload multiple files from different domains to:

  • Find relationships between different domains
  • Connect concepts across fields (e.g., AI + Healthcare)
  • Discover hidden connections in your data
Example:

File 1 (Science): "Einstein developed relativity"

File 2 (Technology): "GPS uses relativity"

Cross-Domain Relation: Einstein → GPS (theory_used_in)
